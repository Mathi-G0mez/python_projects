# Book Store Web Scraper

A beginner-level Python script that scrapes book data from a demo website.  

### What it does
- Gets book titles and prices from [books.toscrape.com](https://books.toscrape.com)
- Saves data to a CSV file

### Tech Used
- Python
- BeautifulSoup
- Requests
- CSV module

### How to Run
1. Run:
   ```bash
